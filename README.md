## Pandoc

Markdown -> HTML / PDF conversion done with [Pandoc](https://pandoc.org/MANUAL.html)

The following command can generate Markdown files to PDF. Requires `wkhtmltopdf`

```
pandoc <input>.md -o <output>.pdf -t html
```
